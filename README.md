# Ai Story Generator

> GPT-2 powered story generator with genre and style control

## Overview
This project is part of a comprehensive AI/ML portfolio covering nlp python.

## Features
- Production-ready implementation
- Well-documented code with examples
- Easy to customize and extend

## Installation

```bash
git clone https://github.com/jaychampaneri14/ai-story-generator.git
cd ai-story-generator
pip install -r requirements.txt
python main.py
```

## Requirements
See `requirements.txt` for full dependencies.
Main libraries: Nlp, Python, Generative Ai

## Usage
```bash
python main.py
```
Run with `--help` for full options.
